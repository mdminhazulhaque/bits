# Bits

My blog, built using [Jekyll](https://github.com/jekyll/jekyll).

## Layout Features

* Sidebar
* Pagination

## Others

* Used [Disqus](https://disqus.com) as comment hosting system
* Auto `sitemap.xml` generator using [jekyll-sitemap](https://github.com/jekyll/jekyll-sitemap)
